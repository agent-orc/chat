ChatRow/History phase ✔ done (5c)

chat-row and the project-chat history (rail + virtualised list incl. load strategy) extracted as the /history entry point; chat-row now uses the library role badge, so there is a single role-badge truth.

Result: the library covers the full chat UI surface
4

Conversation Lab + npm publish 0.1.0 waiting for npm login

The release workflow and scripts/release.sh are in place (commit bea8636), the Conversation Lab demo app ships (d995a7d). One-time manual step left: npm requires the package to exist before a Trusted Publisher can be configured — so log in once (2FA or a granular token with 2FA bypass is mandatory) and run the first publish (npm run build && cd dist/coding-agent-chat && npm version 0.1.0 --no-git-tag-version && npm publish --access public), then register the Trusted Publisher on npmjs.com (repo coding-agent-chat, workflow release.yml) — mirroring the runner's NuGet setup. The @coding-agent scope must belong to you.

Result: installable @coding-agent/chat@0.1.0
5

Host adoption in the Agent Studio frontend ✔ done (5e)

Completed 2026-07-03 (app commit 90a9120b): the frontend consumes @coding-agent/chat (as a file: dependency on dist/ for now, registry version after the first publish). All ~72 duplicated files deleted (−16,644 lines), ~200 consumers moved to library entry points and cac-* selectors, seams wired (provideCodingAgentChat, PROJECT_CHAT_DATA_SOURCE adapter, CHAT_HISTORY_CONFIRM, SignalR via appendLive()). The host keeps, as planned: transport, confirm dialog UI, diff viewer, CLI console, rich editor, attachment upload. Verified: green build, 991 unit tests with zero new failures, two adversarial reviews, live smoke on dev and stable without console errors.

Result: a single source of truth — double maintenance has ended
Remaining manual work (the only open item): the first npm publish — see step 4. Afterwards replace the file: dependency in the app frontend with the registry version. Until then: after every library rebuild (npm run build) run npm install in the app frontend (the file: dependency is a physical copy; documented in frontend/AGENTS.md).

In the library extracted

  • /conversation — cac-conversation-view, ToolBurstChip, SessionCard

  • /markdown — cac-markdown + utils (marked + dompurify, SSR-safe)
  • /history — cac-chat-row, cac-project-chat-list, rail, phase summary
  • /shared — tooltip, stick-to-bottom, lightbox directive + tokens
